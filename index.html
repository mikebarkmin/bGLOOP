Welcome bGLOOP

bGLOOP is a reimplementation of the closed source GLOOP library, a library for teaching beginner courses for OOP in Java with application to 3d programming. It is basically a wrapper for the functionality in jogl. It's main focus is to make creation of small 3d programs really simple to focus on the basic ideas of OOP.

The API is in German.
